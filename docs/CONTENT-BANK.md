# 🧠 بانک محتوای دستی (تا وقتی هوش مصنوعی واقعی وصل نشده)

## چرا این فایل وجود داره؟

الان (۵ ژوئیه ۲۰۲۶) هنوز هیچ کلید API فعالِ هوش مصنوعی (AvalAI/GapGPT/Groq/Gemini) توی این پروژه تنظیم نشده. OpenRouter محدودیت نرخ می‌خوره و DeepSeek موجودی نداره (جزئیات کامل در `docs/LLM-PROVIDERS-2026.md`). یعنی هر بار که فکتوری اجرا می‌شد، اسکریپت‌ها به یه قالب آفلاین بسیار ساده و ژنریک (فقط ۵ خط) سقوط می‌کردن.

شما صریحاً خواستید که تا وقتی خودتون یه کلید هوش مصنوعی اضافه نکردید، **من خودم کار «هوش مصنوعی» رو انجام بدم**. این فایل دقیقاً همونه.

## این چیه؟

`core/content_bank.py` شامل ۱۸ اسکریپت واقعی، کامل (هرکدوم ۱۰-۱۱ صحنه، حدود ۲۷۰-۳۴۰ کلمه) هستش که من شخصاً نوشتم — نه یک قالب ساده، نه چیزی که با `random.choice` انتخاب بشه. هر ادعای واقعیت (تاریخی، علمی، آماری) رو قبل از نوشتن، با جستجوی زنده در وب بررسی و تأیید کردم.

پوشش فعلی:
- **۱۲ اسکریپت انگلیسی**: ۲ تا برای هرکدوم از ۶ نیش (روانشناسی، تاریخ/راز، جنایی، لایف‌استایل لاکچری، مالی، فضا/علم)
- **۶ اسکریپت فارسی**: ۱ تا برای هرکدوم از ۶ نیش (اولویت چون کانال LuxeVista فعلاً انگلیسیه، ولی برای کانال‌های فارسی آینده آماده‌ست)

## ساختار هر اسکریپت (دقیقاً همون ساختاری که از هوش مصنوعی هم خواسته می‌شه)

طبق تحقیق `docs/YOUTUBE-GROWTH-AND-ENGAGEMENT.md`:
1. قلاب قوی در همون خط اول، بدون معرفی کانال
2. بدنه با آرک روایی مشخص (۸-۹ صحنه)
3. یادآوری سبک و زودهنگام برای «دنبال کردن» بعد از اولین نکته‌ی جذاب
4. یک سؤال مشخص و آسان برای پاسخ، وسط ویدیو (نه یه سؤال کلی مثل «نظرتون چیه؟»)
5. بخش پایانی دوکاره: هم یه دلیل مشخص برای دنبال کردن، هم یه سؤال کامنت دوم و متفاوت

## چطور خودکار انتخاب می‌شه؟

هیچ سیم‌کشی اضافه‌ای لازم نبود، چون:
- عنوان‌های استفاده‌شده در `content_bank.py` **کاملاً یکسان** هستن با `evergreen_topics` توی `core/content_config.py`.
- `NicheAnalyzer.analyze_market()` وقتی موضوع ترندِ واقعی پیدا نمی‌کنه (یا کانال قبلاً همه‌شون رو ساخته)، به همین لیست evergreen سقوط می‌کنه — و از این به بعد، **اول یه موضوعی رو انتخاب می‌کنه که براش اسکریپت دستیِ آماده وجود داره**.
- `ScriptWriter.write_script()` هم به همین ترتیب اول LLM واقعی رو امتحان می‌کنه، بعد `content_bank`، و فقط اگه هیچ‌کدوم نبود می‌ره سراغ قالب ژنریک ۵ خطی قدیمی.

نتیجه: همین امروز، بدون هیچ کلید API، فکتوری ویدیوهای واقعی با اسکریپت‌های تحقیق‌شده و باکیفیت می‌سازه، نه یه قالب تکراری و ساده.

## چه زمانی این بانک دیگه استفاده نمی‌شه؟

به محض اینکه یکی از این کلیدها رو اضافه کنید (AvalAI پیشنهاد میشه، توی `docs/LLM-PROVIDERS-2026.md` توضیح داده شده) و کار کنه، `LLMRouter` همیشه **اول** امتحان می‌شه و موفق می‌شه — یعنی اسکریپت‌های دستی این فایل دیگه اصلاً صدا زده نمی‌شن و به‌جاش هر بار یه اسکریپت تازه و منحصربه‌فرد از هوش مصنوعی واقعی تولید می‌شه. هیچ تغییری لازم نیست، این جابجایی خودکاره.

## منابع بررسی‌شده برای هر اسکریپت

| موضوع | نکات کلیدی بررسی‌شده |
|---|---|
| چرا تعلل می‌کنیم | تحقیق دکتر تیم پیچیل (دانشگاه کارلتون) — تعلل مشکل تنظیم هیجانه نه مدیریت زمان؛ نقش آمیگدال؛ اثر «بخشش خود» |
| دوران کودکی و روابط بزرگسالی | مطالعه‌ی طولی دانشگاه مینه‌سوتا روی ۷۰۵ نفر، ۳۰ سال پیگیری؛ نقش کیفیت رابطه با مادر و دوستی‌های کودکی |
| راز کشتی مری سلست | تاریخ واقعی ۱۸۷۲؛ تئوری بخار الکل بر اساس منابع تاریخی معتبر (بریتانیکا، EBSCO) |
| کتابخانه‌ی اسکندریه | تحقیق نشنال جئوگرافیک و AskHistorians — آتش‌سوزی سزار ۴۸ قبل از میلاد، حمله‌ی اورلیان ۲۷۲ میلادی، فرسایش طبیعی پاپیروس |
| قاتل گلدن استیت | دستگیری واقعی جوزف دی‌آنجلو در ۲۰۱۸ با GEDmatch؛ جزئیات از منابع حقوقی و خبری معتبر |
| جعل ون میخرن | ماجرای واقعی هان ون میخرن، محاکمه‌ی ۱۹۴۵، نقاشی زنده در دادگاه برای اثبات جعل |
| جزیره‌های خصوصی میلیاردرها | داده‌های قیمت واقعی از Visual Capitalist (بل آیلند، لاناایی) و پروژه‌ی نیوهلند آبراموویچ |
| صبح میلیاردرها | تحقیق روی ۳۶ میلیاردر خودساخته؛ عادات واقعی بافت، برنسون، ماسک، بزوس |
| ورشکستگی برندگان لاتاری | افشای اینکه آمار «۷۰٪» ساختگیه (NEFE رسماً تکذیب کرده)؛ ردیابی واقعی USA TODAY روی ۳۱ برنده‌ی پاوربال |
| صندوق‌های شاخص | گزارش رسمی SPIVA — ۸۰-۹۵٪ صندوق‌های فعال شکست می‌خورن در ۱۵ سال |
| سقوط به سیاهچاله | فیزیک واقعی اسپاگتی‌شدن، تفاوت سیاهچاله‌ی کوچک و ابرپرجرم، تأخیر زمانی از دید ناظر بیرونی |
| عجیب‌ترین سیاره‌ها | WASP-121b (باران یاقوت)، 55 Cancri e (الماس)، HD 189733b (باران شیشه) — همه با منابع علمی معتبر |

هر ادعا قبل از نوشتن اسکریپت با `web_search` زنده بررسی شد، نه از حافظه یا حدس.
