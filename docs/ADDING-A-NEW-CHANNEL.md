# ➕ اضافه کردن کانال جدید (هر نیچی، هر زبانی)

سیستم کاملاً **config-driven** طراحی شده — یعنی برای هر ترکیب جدید (نیچ × زبان)، کد جدید نمی‌نویسید، فقط تنظیمات اضافه می‌کنید.

## 🚀 روش سریع (کانالی با نیچ/زبان موجود)

```bash
python scripts/add_channel_wizard.py
```

این ویزارد تعاملی از شما می‌پرسه:
1. کدوم نیچ (از لیست موجود)
2. کدوم زبان (از لیست موجود)
3. اسم و شناسه کانال
4. آیا OAuth یوتیوب رو قبلاً تنظیم کردید یا نه

و در نهایت خودش کانال رو در `channels/database.json` ثبت می‌کنه.

## 🌍 زبان‌های موجود الان

فارسی (`fa`)، انگلیسی (`en`)، عربی (`ar`)، اسپانیایی (`es`)، ترکی (`tr`)، آلمانی (`de`)، فرانسوی (`fr`) — همه با صدای رایگان و باکیفیت edge-tts.

**نکته مهم درباره چندزبانگی:** به‌جای اینکه یک اسکریپت انگلیسی بنویسیم و بعد دوبله کنیم، سیستم مستقیماً اسکریپت رو **به زبان مقصد** می‌نویسه (چون Gemini چندزبانه است) — این هم کیفیت بهتر داره (طبیعی‌تر، بدون حس ترجمه‌ای بودن) هم ساده‌تره.

## ➕ اضافه کردن نیچ کاملاً جدید (که در لیست بالا نیست)

فقط یک فایل رو ویرایش کنید: `core/content_config.py`، و در دیکشنری `NICHES` یک بلوک جدید اضافه کنید:

```python
"my_new_niche": {
    "label_fa": "اسم فارسی نیچ",
    "label_en": "English Niche Name",
    "rpm_estimate": "$X-Y",
    "subreddits": ["subreddit1", "subreddit2"],   # برای موضوع‌یابی خودکار
    "search_terms": ["query1", "query2", ...],     # برای جستجوی فوتیج در Pexels/Pixabay
    "evergreen_topics": [                          # موضوعات پیش‌فرض وقتی Reddit/Trends جواب نده
        "موضوع نمونه ۱", "موضوع نمونه ۲", ...
    ],
},
```

همین! بعدش `python scripts/add_channel_wizard.py` رو اجرا کنید، نیچ جدید خودکار توی لیست ظاهر می‌شه.

## ➕ اضافه کردن زبان کاملاً جدید

در همون فایل، دیکشنری `LANGUAGES` رو ویرایش کنید:

```python
"it": {"label": "Italiano", "voice": "it-IT-DiegoNeural", "voice_alt": "it-IT-IsabellaNeural"},
```

برای پیدا کردن اسم دقیق صداها:
```bash
python -m edge_tts --list-voices | grep it-IT
```

## 📋 راهنمای کامل قدم‌به‌قدم برای یک کانال کاملاً جدید

1. کانال یوتیوب رو در youtube.com بسازید (اگه نساختید)
2. `python scripts/setup_youtube_oauth.py` رو اجرا کنید (یک‌بار، با مرورگر) — راهنما در `docs/YOUTUBE-OAUTH-SETUP.md`
3. Refresh token که گرفتید رو در GitHub Secrets بذارید
4. `python scripts/add_channel_wizard.py` رو اجرا کنید و کانال رو ثبت کنید
5. تست بدون آپلود: `SKIP_UPLOAD=1 TARGET_MINUTES=2 python main.py`
6. اگه راضی بودید: `python main.py` (یا صبر کنید تا ورک‌فلوی روزانه خودکار اجرا بشه)

هر تعداد کانال، با هر ترکیبی از نیچ و زبان، می‌تونید اضافه کنید — همه در یک اجرای `main.py` (یا یک اجرای روزانه GitHub Actions) پردازش می‌شن.
