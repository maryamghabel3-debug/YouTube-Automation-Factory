# ➕ اضافه کردن کانال جدید (هر نیچی، هر زبانی)

سیستم کاملاً **config-driven** طراحی شده — یعنی برای هر ترکیب جدید (نیچ × زبان)، کد جدید نمی‌نویسید، فقط تنظیمات اضافه می‌کنید.

## 🚀 روش اصلی و توصیه‌شده: چت با FactoryBot توی تلگرام

از این به بعد **نیازی به باز کردن گیت‌هاب یا اجرای اسکریپت از کامپیوتر نیست** — همه‌چیز از توی تلگرام (حتی از گوشی) انجام می‌شه:

```
/newchannel
```

بات ازتون می‌پرسه:
1. کدوم نیچ (عدد رو انتخاب کنید)
2. کدوم زبان
3. اسم نمایشی کانال
4. شناسه کوتاه داخلی (یا `-` برای پیشنهاد خودکار)

و خودش کانال رو در `channels/database.json` ثبت می‌کنه و پوش می‌کنه — کاملاً بدون دست زدن به گیت‌هاب.

سپس برای وصل کردن اکانت واقعی یوتیوب:
```
/oauth شناسه_کانال
```
بات یک لینک + کد کوتاه بهتون می‌ده؛ اون لینک رو با هر مرورگری (حتی همون گوشی) باز می‌کنید، کد رو وارد می‌کنید، با اکانت گوگلی مالک کانال وارد می‌شید و اجازه می‌دید. بات هر چند دقیقه یک‌بار خودش چک می‌کنه و به محض تأیید:
- توکن رو **خودش** به‌صورت رمزنگاری‌شده در GitHub Secrets ذخیره می‌کنه (نیازی به کپی/پیست دستی نیست)
- خودش خط جدید رو در ورک‌فلوی `run-factory.yml` اضافه می‌کنه
- بهتون پیام می‌ده که وصل شد ✅

جزئیات فنی این مکانیزم (device authorization grant) در `docs/YOUTUBE-OAUTH-SETUP.md` توضیح داده شده — فقط **یک‌بار** باید یک OAuth Client از نوع خاص در Google Cloud Console بسازید (این کار محدود به کل پروژه‌ست، نه هر کانال).

برای تست سریع (بدون آپلود واقعی، فقط ببینید محتوا چطور درمیاد):
```
/testvideo شناسه_کانال
```
و برای ساخت + آپلود واقعی:
```
/makevideo شناسه_کانال
```

دستورات کامل بات: `/help`

## 🛠 روش جایگزین (خط فرمان، برای کسی که به کامپیوتر/سرور دسترسی داره)

```bash
python scripts/add_channel_wizard.py
```

این ویزارد تعاملی همون کار بات رو از خط فرمان انجام می‌ده و در نهایت خودش کانال رو در `channels/database.json` ثبت می‌کنه.

## 🌍 زبان‌های موجود الان

فارسی (`fa`)، انگلیسی (`en`)، عربی (`ar`)، اسپانیایی (`es`)، ترکی (`tr`)، آلمانی (`de`)، فرانسوی (`fr`) — همه با صدای رایگان و باکیفیت edge-tts.

**نکته مهم درباره چندزبانگی:** به‌جای اینکه یک اسکریپت انگلیسی بنویسیم و بعد دوبله کنیم، سیستم مستقیماً اسکریپت رو **به زبان مقصد** می‌نویسه (چون Gemini چندزبانه است) — این هم کیفیت بهتر داره (طبیعی‌تر، بدون حس ترجمه‌ای بودن) هم ساده‌تره.

## ➕ اضافه کردن نیچ کاملاً جدید (که در لیست بالا نیست)

فقط یک فایل رو ویرایش کنید: `core/content_config.py`، و در دیکشنری `NICHES` یک بلوک جدید اضافه کنید:

```python
"my_new_niche": {
    "label_fa": "اسم فارسی نیچ",
    "label_en": "English Niche Name",
    "rpm_estimate": "$X-Y",
    "subreddits": ["subreddit1", "subreddit2"],   # برای موضوع‌یابی خودکار
    "search_terms": ["query1", "query2", ...],     # برای جستجوی فوتیج در Pexels/Pixabay
    "evergreen_topics": [                          # موضوعات پیش‌فرض وقتی Reddit/Trends جواب نده
        "موضوع نمونه ۱", "موضوع نمونه ۲", ...
    ],
},
```

همین! بعدش توی تلگرام `/newchannel` رو بزنید (یا `python scripts/add_channel_wizard.py`)، نیچ جدید خودکار توی لیست ظاهر می‌شه.

## ➕ اضافه کردن زبان کاملاً جدید

در همون فایل، دیکشنری `LANGUAGES` رو ویرایش کنید:

```python
"it": {"label": "Italiano", "voice": "it-IT-DiegoNeural", "voice_alt": "it-IT-IsabellaNeural"},
```

برای پیدا کردن اسم دقیق صداها:
```bash
python -m edge_tts --list-voices | grep it-IT
```

## 📋 راهنمای کامل قدم‌به‌قدم برای یک کانال کاملاً جدید (روش تلگرامی)

1. کانال یوتیوب رو در youtube.com بسازید (اگه نساختید)
2. توی تلگرام: `/newchannel` → نیچ/زبان/اسم رو بدید
3. `/oauth شناسه_کانال` → لینک+کد رو بگیرید، با اکانت گوگلی مالک کانال تأیید کنید — بات خودش بقیه رو انجام می‌ده
4. تست بدون آپلود: `/testvideo شناسه_کانال`
5. اگه راضی بودید: `/makevideo شناسه_کانال` یا صبر کنید تا اجرای روزانه خودکار (کرون) پیش بره

هر تعداد کانال، با هر ترکیبی از نیچ و زبان، می‌تونید اضافه کنید — همه در یک اجرای روزانه GitHub Actions پردازش می‌شن.
